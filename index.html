Altroz - News Magazine HTML Template with of features options. It is a great solution for a publishers, large magazines, newspapers, bloggers and other news websites. Use Altroz for many niches, such as fashion, gaming, videos, music, business and world news. This HTML5 template based on the latest Bootstrap 4 framework and it’s fully responsive.

Page head contains Title, Metadata, Favicon and CSS.

Fonts :-

Altroz is a News Magazine HTML Template uses Open Sans font for headings from Google Web Fonts. You are able to choose more than 600+ fonts from Google Web Fonts. The font imported in the theme css file

In style project you will find it in stylesheet.css file you can find it in

 // Font Importing 
 
    <link rel="stylesheet" href="css/font-awesome.min.css">
	
    <!-- Google Fonts -->
    <link href="https://fonts.googleapis.com/css?family=Nunito:300,400,500,600,700,800&display=swap" rel="stylesheet">
